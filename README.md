# pingpong-project

一个可在电脑和手机浏览器使用的「乒乓球失误记录」应用，已升级到 v2：支持后台数据库存储，并提供微信小程序端代码。

## v2 功能
- 网页端记录比赛时间、回合阶段、失误大类和细分失误。
- 失误细分基于较明确、可针对改进的技术/战术描述。
- 后端 API + SQLite 持久化（关闭浏览器后数据不丢失）。
- 自动生成基础统计：总数、最高频大类、最高频细分。
- 提供微信小程序 `miniprogram/` 端代码，可直接对接同一 API。

## 目录
- `index.html` `styles.css` `app.js`: 网页端
- `server.py`: 后端 API（Python 标准库 + SQLite）
- `miniprogram/`: 微信小程序代码

## 启动方式
### 1) 启动后端
```bash
python3 server.py
```
默认地址：`http://127.0.0.1:8787`

### 2) 启动前端静态页面
```bash
python3 -m http.server 4173
```
访问：`http://127.0.0.1:4173`

网页端会优先连接后端 API，连接成功则使用云端数据库；失败时自动回退到浏览器本地存储。

## 微信小程序说明
1. 下载并打开微信开发者工具。
2. 导入 `miniprogram/` 目录。
3. 修改 `miniprogram/config.js` 中的 `API_BASE` 为你的后端公网地址（或可访问的局域网地址）。
4. 在开发者工具「详情-本地设置」中开启“不校验合法域名”（本地调试阶段）。
5. 运行即可进行记录和统计。
